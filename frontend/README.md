## 🎯 O Objetivo
A FEAPP atua desde 2000 como um órgão aglutinador, organizando grandes eventos (como a Vila Julina) e apoiando dezenas de entidades. O desafio deste projeto foi criar uma vitrine digital que:
1.  **Centralizasse as informações** das 33 entidades federadas.
2.  **Aumentasse a transparência** e credibilidade da instituição.
3.  **Facilitasse a doação** e o engajamento de novos voluntários através de uma UX otimizada.

---

## ✨ Funcionalidades Principais

* **⚡ Hero Section de Impacto:** Apresentação clara da missão com Call-to-Actions (CTAs) estratégicos.
* **🏢 Diretório de Entidades:** Listagem organizada das instituições, com filtros por área de atuação (Idosos, Crianças, Saúde, etc.) e links diretos para contato.
* **🎉 Gestão de Eventos:** Destaque para os eventos sazonais (Vila Julina, Vila Natal) que movem a economia solidária da cidade.
* **📱 Design Responsivo:** Layout fluido que se adapta perfeitamente a celulares, tablets e desktops.
* **🎨 UI Moderna:** Uso de componentes acessíveis e animações suaves para retenção do usuário.

## 🛠️ Tech Stack (Tecnologias)

O projeto foi construído utilizando as melhores práticas do ecossistema moderno de Frontend:

| Categoria | Tecnologia | Por que foi escolhida? |
| :--- | :--- | :--- |
| **Core** | **React + TypeScript** | Robustez, tipagem estática e componentização escalável. |
| **Build** | **Vite** | Ambiente de desenvolvimento ultra-rápido e build otimizado. |
| **Estilo** | **Tailwind CSS** | Estilização utilitária para desenvolvimento ágil e consistente. |
| **UI Kit** | **shadcn/ui** | Componentes de alta qualidade, acessíveis e customizáveis. |
| **Ícones** | **Lucide React** | Biblioteca de ícones leve e consistente visualmente. |
| **Animação** | **Framer Motion** | Micro-interações e transições fluidas para melhor UX. |

## 🚀 Como Executar Localmente

Siga os passos abaixo para rodar o projeto na sua máquina:

### Pré-requisitos

* Node.js (versão 18 ou superior)
* Gerenciador de pacotes (NPM, PNPM ou Yarn)

### Passo a Passo

1. **Clone o repositório**
```bash
git clone [https://github.com/MatheusHB018/remix-of-rede-do-bem-feapp.git](https://github.com/MatheusHB018/remix-of-rede-do-bem-feapp.git)

```


2. **Instale as dependências**
```bash
cd remix-of-rede-do-bem-feapp
npm install
# ou pnpm install

```


3. **Inicie o servidor de desenvolvimento**
```bash
npm run dev

```


4. **Acesse no navegador**
O projeto estará rodando em: `http://localhost:5173`
